2011, BOAS PRATICAS DE USO DOS DADOS GOVERNAMENTAIS ABERTO PELA SOCIEDADE.

ABSTRAÇÃO

Este documento tenta reunir boas práticas na manipulação de dados governamentais abertos pela sociedade, e é recomendado que você saiba o que são estes dados antes de continuar lendo este texto. Existe um outro documento que escrevi sobre o que são dados abertos governamentais, que é 2011-Publicidade-de-dados-governamentais.html.

Manipular dados sempre envolve algumas atenções que devem ser tomadas para evitar trabalhos desnecessários e também para que o esforço realizado se torne realmente produtivo e construtivo.

O uso de dados abertos pela sociedade, é uma tarefa que pode com certeza deixar o mundo melhor. Porém, não é uma tarefa simples disponibilizar os dados, e nem oferecer sempre o significado destes dados, e por isto em algumas situações isto pode dificultar o trabalho de manipulação dos dados.

Neste documento espero realizar uma visão geral de como aproveitar melhor os dados oferecidos pelo governo em formatos abertos.
